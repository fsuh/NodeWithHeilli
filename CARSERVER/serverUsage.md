# CarServer Usage

## get all cars
```
http://localhost:3000/cars
```

## get all types
```
http://localhost:3000/cartypes
```

## Search by License
```
http://localhost:3000/search/bylicense?value=ABC-1
```

## search by model
```
http://localhost:3000/search/bymodel?value=Hoppa
```
